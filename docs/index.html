About This Portfolio

Professional Self-Assessment

This ePortfolio represents the progression of skills developed throughout the Computer Science program and demonstrates the ability to design, build, and improve real-world computing solutions. Coursework and hands-on projects strengthened technical foundations in software engineering, algorithms, databases, and secure development practices, while also shaping a forward-looking perspective on the evolving role of software engineers in an AI-driven environment.

Experience across multiple projects contributed to an understanding of collaborative development, even when working independently. Development work required thinking through how solutions would be maintained, extended, and understood by others, which aligns with real-world team environments. Communication skills were strengthened through writing technical documentation, structured narratives, and explaining design decisions in a clear and accessible way for both technical and non-technical audiences.

Work with data structures and algorithms is demonstrated through the reinforcement learning maze agent, where problem-solving, optimization, and iterative improvement were required to influence model behavior. This artifact highlights the ability to apply algorithmic thinking, evaluate performance, and adjust parameters such as exploration rate, reward shaping, and training constraints to improve outcomes.

Software engineering and database skills are reflected in the Event Tracking Android application. Enhancements included improving login workflows, implementing validation, refining database interactions, and strengthening the overall structure of the application. These improvements demonstrate an understanding of application architecture, data persistence, and user experience considerations.

Security concepts were incorporated through validation logic, credential handling, and the implementation of a more realistic password reset approach. This reflects a growing awareness of how to anticipate risks, reduce vulnerabilities, and design systems that better protect user data.

The artifacts included in this portfolio collectively demonstrate the ability to analyze existing systems, identify weaknesses, and implement meaningful improvements. The software engineering artifact highlights application design and usability, the algorithms artifact demonstrates problem-solving and model tuning, and the database artifact reflects data integrity and secure handling practices. Together, these artifacts represent a well-rounded foundation in computer science and readiness to contribute to modern, evolving technology environments.
